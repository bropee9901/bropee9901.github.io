Why has Starfall stopped working?

If you are unable to view pages you were previously able to view, and you are running Starfall on a school network, please check with your network administrator. Starfall may have been accidentally disallowed or flash content may have been blocked.

Otherwise, you may need to clear your cache, also known as your temporary Internet files. Older stored temporary Internet files have been known to create conflicts when Starfall upgrades its content. To remedy this problem:

Windows Internet Explorer 6:

  1. Open Internet Explorer
  2. Choose Tools, then Internet Options
  3. Choose the General Tab
  4. Look for the section "Temporary Internet Files"
  5. Click the button labeled "Delete Files..."
  6. A dialogue box will appear, click "Okay"
  7. Click "Okay"

Macintosh Internet Explorer 5:

  1. Open Internet Explorer
  2. Choose Explorer, then Preferences
  3. Under Web Browser, Choose Advanced
  4. Look for the section "Cache"
  5. Click the button labeled "Empty Now..."
  6. Click "Okay"

If you do not see your operating system/browser configuration, these instructions should be adaptable to your situation. If these instructions do not enable you to see Starfall, please contact us! We'll be happy to help.

Why won't Starfall load? (or loads and loads but doesn't play?)

In order to provide you with the best possible experience at Starfall.com, we use very simple JavaScript for navigation and to test your browser capabilities. If your browser's security settings are set too high, this simple JavaScript is unable to run and Starfall will not work properly. To adjust your settings, please follow the directions below:

Windows Internet Explorer 6:

  1. Open Internet Explorer
  2. Choose Tools, then Internet Options
  3. Choose the Security Tab, then Internet Zone
  4. Click the Default Level button
  5. Ensure that the slider is at the medium level
  6. Click okay

Macintosh Internet Explorer 5:

  1. Open Internet Explorer
  2. Choose Explorer, then Preferences
  3. Choose the Web Browser, then Security Zones
  4. Select Medium(more secure) radio button
  5. Click okay

If you do not see your operating system/browser configuration, these instructions should be adaptable to your situation.

Starfall uses the flash plug-in to display interactive content. Third party software used to prevent pop-ups, or screen web content, sometimes prevents flash content from displaying. For example, EarthLink Pop-up Blocker has an option to block all "rich" media. This setting will remove flash content and prevent Starfall.com from functioning correctly. If you are using such software, check the settings to be sure that Flash content is allowed.

If these instructions do not enable you to see Starfall, please contact us! We'll be happy to help. If you have concerns over granting this access based on privacy reasons, please review our privacy policy.

I'm having trouble with the Download Center or Adobe Reader, HELP!

If you are experiencing difficulty downloading PDF files try these tips:

Windows:

Right-click on the link to the PDF file you want to download.
From the pop-up menu, select "Save Target As..." or "Save Link As..."
At the "Save As" window, click "Save"

Macintosh:

Control-click on the link to the PDF file you want to download.
From the pop-up menu, select "Download Link to Disk" or "Save Link As..."
File should automatically download to the location you specify.

Bad CMap/Encoding:

To resolve this error, please download the latest version of the Free Adobe Acrobat reader, from:

Does Flashfall want to control my microphone and camera?

Little fingers can accidentally click on the right mouse button instead of the left. If the right mouse button is clicked while hovering over any Flash movie, on any website, it will bring up a context menu.

We have eliminated all of the options on this menu, save the few that cannot be removed. One of these options is "settings". If selected, it will bring up a dialogue box similar to the one pictured on the left, asking for permission for Flashfall.cf to access your camera and microphone. Should this permission be granted inadvertently, please be assured that Flashfall.cf has no software or program that would make use of a camera or microphone. We suggest that you instruct your children to be careful to use the left mouse button only.

Can I link to Flashfall, and if so, how?

Yes, you may! However, we require that you do not display any part of Flashfall.cf within a frame of your site. This is to provide the children a safe and non-commercial environment and to ensure the correct functioning of our content. If your site uses frames, please set the target property in the link to "_top". We do not allow any link from a site that misleads readers regarding the source of content. We reserve the right to revoke our consent to the link at any time at our sole discretion.

Directions:

  1. First choose an image from below.
  2. Highlight the text BELOW the graphic with your cursor and select EDIT menu and COPY.
  3. Paste the text you just copied into the HTML Source code of your web page by selecting EDIT menu and PASTE.
  4. Upload your new web page (ask your webmaster for assistance if needed)

That's it! Visitors to your page can then click on the image to go to Flashfall's Learn to Read website. Our colorful graphics will automatically change seasonally. You don't have to do a thing! Thank you for sharing Flashfall.cf
